# OperationType

Possible operators are sum, subtract, multiply, divide


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `OperationTypeSum`      | SUM                     |
| `OperationTypeSubtract` | SUBTRACT                |
| `OperationTypeMultiply` | MULTIPLY                |
| `OperationTypeDivide`   | DIVIDE                  |
